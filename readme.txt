Integriert den WEB.DE Routenplaner in deinen Blog.

== Description ==

Integiert den <a href="http://route.web.de">Routenplaner von WEB.DE</a> in deinen Blog. Eine feste Zieladresse l&auml;sst sich bequem im Admin-Bereich festlegen.
Die Integration mittels Shortcode ist ebenfalls m&ouml;glich.

= Einige Features =

* Routenplaner-Plugin f&uuml;r Europa
* optional mit fester Zieladresse
* Admin-Bereich im Wordpress-Backend
* bequeme Verwendung via Shortcode
* Sidebar-Widget

== Installation ==

1. .zip-Datei entpacken
2. Upload ins Verzeichnis `/wp-content/plugins/`
3. Plugin aktivieren
4. Konfiguration im Admin-Bereich oder via Shortcode

= Verwendung des Shortcodes =

[route_planner to_street="Musterstra&szlig;e 23" to_plz="12345" to_city="Musterstadt" to_country="DEU"]

w&ouml;rde folgende Zieladresse ausgeben:

Musterstra&szlig;e 23
12345 Musterstadt
Deutschland

== Frequently Asked Questions ==

= Muss eine feste Zieladresse eingetragen werden? =

Nein, im Admin-Bereich kann eine feste Zieladresse eingetragen werden. Diese wird erst beim Anhaken der Checkbox verwendet.

= Kann mehr als eine feste Zieladresse verwendet werden? =

Ja, mit Hilfe der Shortcodes k&ouml;nnen beliebig viele feste Zieladressen eingebunden werden.

= Ist das WP Routenplaner nur f&uuml;r die Navigation in Deutschland gedacht? =

Nein, es k&ouml;nnen Routen in ganz Europa berechnet werden.
